# Hinweis

Diese Aufgabe stammt aus dem Selbsttest des Kurses "Anwendungsnahes Programmieren mit Python" von Markus Dahinden und Lukas Fässler.

Der eingesetzte Testrunner basiert auf dem Unit-Test Framework. Er nutzt TAP Output zur Darstellung des Resultats. Er ist nicht speziell für den Prüfungseinsatz gehärtet (er läuft im gleichen Benutzerkontext wie das Programm der Studierenden). Dementsprechend können diese das Laufzeitverhalten des Testrunners grundlegend verändern). Es wird daher empfohlen, die Studierendenprojekte nicht vollautomatisch, sondern kriterienbasiert manuell zu korrigieren.
